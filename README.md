# AI Climate-Smart Agriculture Starter (Kenya)

A schema-first starter kit for building explainable and practical AI tools supporting climate-smart agriculture and food security in Kenya.

## Why this exists (Feb 2026)
Climate variability, pest outbreaks, and weather unpredictability are increasing risks for smallholder farmers. This repository provides structured data schemas, evaluation rubrics, and baseline analytics to support responsible AI adoption.

## What this repo contains
- Farm, soil, pest, and weather data schemas
- Synthetic sample datasets
- Baseline forecasting examples
- Evaluation rubric for real-world usefulness
- Privacy and farmer consent guidance
- Deployment checklist (offline-first and SMS-ready)
